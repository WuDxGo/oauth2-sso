### 方式一：快速启动（推荐）

按照 [QUICKSTART.md](QUICKSTART.md) 的指引，5 分钟即可启动系统：

```bash
# 1. 初始化数据库
mysql -u root -p < init.sql

# 2. 确认 Redis 运行
redis-cli ping

# 3. 编译项目
build.bat

# 4. 启动所有服务
start-all.bat

# 5. 访问测试
浏览器打开：http://localhost:8081
登录账号：admin / 123456
```

## 📝 API 接口示例

### 订单接口（通过网关）
```bash
# 获取所有订单
GET http://localhost:8081/api/orders/orders

# 创建订单
POST http://localhost:8081/api/orders/orders
Content-Type: application/json
{
  "amount": 200.00,
  "description": "测试订单"
}
```

### 用户接口（通过网关）
```bash
# 获取所有用户
GET http://localhost:8081/api/users/users

# 获取当前用户
GET http://localhost:8081/api/users/users/me
```
