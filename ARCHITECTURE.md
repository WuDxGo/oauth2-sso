## JWT Token 结构

```
┌─────────────────────────────────────────────────────────────┐
│                      JWT Token                               │
│  Header.Base64.Payload.Base64.Signature                      │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  Header (头部)                                               │
│  {                                                           │
│    "alg": "RS256",          // 签名算法                     │
│    "typ": "JWT"             // 令牌类型                      │
│  }                                                           │
│                                                              │
│  Payload (负载)                                              │
│  {                                                           │
│    "sub": "admin",          // 主题（用户名）                │
│    "iat": 1234567890,       // 签发时间                      │
│    "exp": 1234575090,       // 过期时间（2 小时后）            │
│    "iss": "http://localhost:8080", // 签发者                │
│    "aud": "gateway-client", // 受众                          │
│    "scope": "read write",   // 权限范围                      │
│    "authorities": ["ROLE_ADMIN"] // 角色权限                │
│  }                                                           │
│                                                              │
│  Signature (签名)                                            │
│  HMACSHA256(                                                 │
│    base64UrlEncode(header) + "." + base64UrlEncode(payload), │
│    RSA_PRIVATE_KEY                                           │
│  )                                                           │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 安全过滤器链

### OAuth2 认证服务器（8080）
```
HTTP Request
    │
    ▼
┌─────────────────────────────────┐
│ SecurityFilterChain (Order 1)   │  ← OAuth2 专用过滤器链
│  - OAuth2AuthorizationEndpoint  │
│  - OAuth2TokenEndpoint          │
│  - OIDC Logout Endpoint         │
└─────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────┐
│ SecurityFilterChain (Order 2)   │  ← 默认表单登录
│  - Form Login                   │
│  - Logout                       │
│  - CSRF Protection              │
└─────────────────────────────────┘
```

## 权限控制模型（RBAC）

```
用户 (User)
   │
   │ 一对多
   ▼
角色 (Role) ────► 权限 (Permission)
   │                │
   │ 多对多          │ 资源类型
   │                │ - 菜单
   │                │ - 按钮
   │                │ - 接口
   ▼                ▼
ADMIN           read
USER            write
```

**示例**:
- admin 用户 → ADMIN 角色 → read + write 权限
- user 用户 → USER 角色 → read 权限
