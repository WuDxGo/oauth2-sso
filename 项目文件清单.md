## 🎯 关键配置项

### application.yml 配置要点

每个服务的配置文件都包含：
- 服务器端口
- 数据源配置（MySQL）
- Redis 配置
- MyBatis 配置
- 日志级别配置
- OAuth2/JWT 配置

### 预配置 OAuth2 客户端

1. **gateway-client**
   - 授权类型：authorization_code
   -  scopes: openid, profile, read, write

2. **order-service**
   - 授权类型：client_credentials
   -  scopes: read, write

3. **user-service**
   - 授权类型：client_credentials
   -  scopes: read, write
